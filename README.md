
# pixo

SVG icon library management and optimization

```sh
npm i -g pixo
```

## CLI

Create a new library

```sh
pixo init
```

Add an icon

```sh
pixo add src/my-icon.svg
```

Add multiple icons

```sh
pixo add src
```

List icons by name

```sh
pixo ls
```

Exports icons as SVG

```sh
pixo export dist
```
